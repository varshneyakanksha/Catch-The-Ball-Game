
# 🎮 Catch The Ball Game

A fun and interactive web game where the player catches a moving ball to score points within a limited time.

## 🚀 Live Demo

👉 npx plugins add vercel/vercel-plugin

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

## 🎯 Features

* Timer-based gameplay
* Increasing challenge
* High score tracking (localStorage)
* Smooth animations

## 💡 How to Play

* Click on the moving ball
* Score as many points as possible before time runs out
